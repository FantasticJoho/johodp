üöÄ Quick Start

Follow these steps to set up a complete OAuth2 flow:

  1. Create a client: POST /api/clients
  2. Create a tenant: POST /api/tenant
  3. Link them: PUT /api/clients/{id}
  4. Register user: POST /api/users/register
  5. Activate account: POST /api/auth/activate
  6. Start OAuth2 flow: GET /connect/authorize
